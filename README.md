# Application description

Приложение написано с использованием Tailwindcss, в качестве store используется Recoil. Присутствует базовый роутинг в навигации и на карточках товара (Routing app). Кроме это используется фейковая регистрация пользователя на сайте, дающая доступ к панели администратора (login: admin, password: admin). Для приложения погоды используется API OpenWeather и API ip-api для получения геопозиции. Для приложения роутинга используется fakestoreapi API чтоб отобразить товары.
